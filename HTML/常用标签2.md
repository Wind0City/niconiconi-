# 稍高级的标签
## 表格标签
基本语法：
```html
    <table>
        <tr>
        <th>表头单元格</th>
        </tr>
        <tr>
            <td>单元格内的文字</td>
            ```
        </tr>
    </table>
```
注释： tr用来定义行 td用来定义单元格，td必须放在tr中，td中间放表格的数据
th为表头单元格，会加粗并且居中显示
属性：(记住相关英文单词即可),要写在table里面
|    属性名    | 属性值            | 对应内容                 |
| :----------: | ----------------- | ------------------------ |
|    alien     | left,center,right | 对齐方式                 |
|    border    | 1或""             | 边框                     |
| cellpadding  | 像素值            | 单元格与其内容的间距     |
| cellspacing  | 像素值            | 单元格与单元格之间的间距 |
| width/height | 像素值或百分比    | 表格的宽度和高度         |
***单元格里可以放任何元素***
## 表格结构标签
将表格分为表头和表格两个内容，还是要在table里面写
表头：`<thead></thead>` 内容:`<tbody></tbody>`
## 合并单元格
方式： 跨行合并：rowspan=“合并个数”  跨列合并 colspan=“合并的单元格个数”
代码要写在目标单元格：跨行最上侧，跨列最左侧
## 列表标签
主要用来布局，分为 无序列表，有序列表，自定义列表

### 无序列表
列表要夹在`<ul>`标签内
用`<li>`标签来定义每个列表的内容
### 有序列表
列表要夹在`<ol>`标签内
仍是用`<li>`标签来定义每个列表的内容
### 自定义列表
常用来对于术语和名词进行解释和描述
列表夹在`<dl>`标签内
内容用`<dt>`标签来描述名词
内容用`<dd>`标签来描述名词解释
## 表单标签（就是填的表
作用：用来收集用户信息
组成：表单域，表单控件，提示信息
### 表单域（一个包含表单元素的区域
用`<form>`双标签来定义
```html
    <form action="发送到的地址" method="提交方式" name="表单域的名字">
    </form>
```
### 表单元素（填写的那些框
1. input输入表单元素
   语法规范`<input type="属性值" />`
   type的属性值，常用的几个重要的属性值
   | 属性值   | 控制的对象                                               |
   | -------- | -------------------------------------------------------- |
   | text     | 文本框(输入文本                                          |
   | password | 密码框(会掩码                                            |
   | radio    | 单选按钮                                                 |
   | checkbox | 定义复选框（选多个                                       |
   | reset    | 定义重置按钮                                             |
   | submit   | 发送数据，相当于提交按钮                                 |
   | button   | 定义按钮，一般是执行某种功能（例如发送验证码，搭配js使用 |
   | file     | 进行文件和图像的上传（选择文件                           |
   | hidden   | 定义隐藏的输入字段                                       |
   其他的属性
   | 属性值    | 控制的对象                                                           |
   | --------- | -------------------------------------------------------------------- |
   | name      | 命名表单元素（相同的名字才是复选按钮，单选和复选框都要有相同的name值 |
   | vale      | 表单的值（即提醒信息，主要针对后台人员                               |
   | checked   | 选择表单默认选择的选项                                               |
   | maxlength | 最多输入的字符长度                                                   |
2. `<lable>`理解为注释标签
   用于绑定一个表单元素，点击这个，就是点击绑定的那个表单元素
   `<lable for="text">用户名：</lable><input type="text" id="text">`
   点击用户名这个名字会进行输入(for里面的名字要有对应的id)
3. 表单控件
   select选择表单元素，（让用户选择多个元素 
   ```html
   <select>
    <option>选项1</option>
    <option>选项2</option>
    <option selected=“selected”>选项3</option>//默认选择选项3
   </select>
   ```
   文本域表单元素（可以写大量文字的文本框
   `<textarea>`双标签来定义
   cols：定义文本框的列数 rows：定义行数
